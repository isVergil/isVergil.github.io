189.旋转数组

2019-01-10
算法

目标

给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

示例 1:

输入: [1,2,3,4,5,6,7] 和 k = 3

输出: [5,6,7,1,2,3,4]

解释:

向右旋转 1 步: [7,1,2,3,4,5,6]

向右旋转 2 步: [6,7,1,2,3,4,5]

向右旋转 3 步: [5,6,7,1,2,3,4]

###示例 2:###

输入: [-1,-100,3,99] 和 k = 2

输出: [3,99,-1,-100]

解释:

向右旋转 1 步: [99,-1,-100,3]

向右旋转 2 步: [3,99,-1,-100]

###C语言实现###

  • 常规思路

    数组依次向后移 k 次

    void rotate(int* nums, int numsSize, int k) {    
    if(k==0)
        return ;
    k %= numsSize;
    //k取余减少移动次数
    int i=1;
    while(i<=k){
        int temp=nums[numsSize-1];
        for(int j=numsSize-1;j>0;j--)
            nums[j]=nums[j-1];
        nums[0]=temp;
        i++;
    }
}

    总结： for 循环里每次交换都要翻转 n 次，总的翻转次数为 kn，时间复杂度为 o(kn) 。

  • 翻转数组

    1、先将数组整个翻转，即 [n,n-1,n-2,…,3,2,1]

    2、再将前 k 个元素翻转，即 [n-k+1,n-k+2,…,n-1,n,n-k,n-k-1,…,3,2,1]

    3、最后将后 n-k+1 个元素翻转，即 [n-k+1,n-k+2,…,n-1,n,1,2,3,…,n-k-1,n-k] 

    void rotate(int* nums, int numsSize, int k) {
    if(k==0)
        return ;
    k %= numsSize;
    reverse(nums,0,numsSize-1);
    reverse(nums,0,k-1);
    reverse(nums,k,numsSize-1);
}
//翻转函数
void reverse(int* nums, int first, int last){
    while(first<last)
    {
        int temp=nums[first];
        nums[first++]=nums[last];
        nums[last--]=temp;
    }
}

    总结： 调用三次翻转函数，第一次翻转 n/2 次，第二次 k/2 次，第三次 (n-1-k)/2 次，总共 n 次，时间复杂度为 o(n) 。
